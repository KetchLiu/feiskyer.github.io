基于 [Hexo](https://hexo.io/zh-cn/docs/) 的个人博客 <https://feisky.xyz>。

## hexo使用方法

创建博客

```
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
```

创建文章

```
hexo new "Hello Hexo"
```

生成静态文件

```
hexo generate
```

发布

```
hexo deploy
```

启动本地服务

```
# http://localhost:4000/
hexo server
```
